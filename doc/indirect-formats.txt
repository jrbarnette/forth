execution token: pointer to definition
definition: function pointer followed by data

definition formats
    general
	fn-ptr, data ...

    primitive
	fn-ptr

    : definition
	do_colon, instr ..., ' EXIT

    CREATE definition
	do_create, xt, byte ..., pad

    CONSTANT definition
	do_constant, cell

    VARIABLE definition
	do_variable, cell

instr formats:
    xt
    do_literal xt, cell
    do_skip xt, offset
    do_fskip xt, offset
    do_s_quote xt, len cell, byte * len, pad
    do_c_quote xt, len byte, byte * len, pad

initialization instruction format
    fn-ptr
    do_literal, cell
    create_name, string, fn-ptr, flags
    tick_name, string


====
indirect VM

instr formats:
    fn-ptr
    do_literal, cell
    do_skip, offset
    do_fskip, offset
    do_s_quote, len, byte * len
    do_c_quote, count, byte * count

: definition
    vm-code-ptr, xt, ...


interp loop
    while (...)
	ip = (*ip)(ip, vmptr);

special instructions -
    NEXT (indirect threaded):
	T = *VIP
	VIP++
	DP = T + 1
	goto *T

    DO-COLON (indirect threaded):
	RPUSH VIP
	VIP = DP
	NEXT

    EXIT (indirect threaded)
	RPOP VIP
	NEXT

    DO-VARIABLE (indirect threaded):
	PUSH DP
	NEXT
